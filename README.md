# webpack-react

Boilerplate webpack-react

## Get-Started

```bash
$ cd C:\Users\{REPLACE_USER}\Documents\GitHub
```

```bash
$ git clone webpack-boilerplate/webpack-react
```

```bash
$ npm i
```

## Usage

```bash
$ npm i
```

### Server [Development]

```bash
$ npm run serv
```

### Build [Development]

```bash
$ npm run dev
```

### Build [Production]

```bash
$ npm run prod
```

## Advanced

#### Eslint [Lint]

```bash
$ npm run lint
```

#### Jest [Tests Units]

```bash
$ npm run test
```
